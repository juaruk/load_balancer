\doxysubsection{Detailed Description}
Entry point for simulating a load balancer with multiple web servers. 

This program prompts the user to enter the number of servers and the total simulation time (in clock cycles). It initializes a load balancer, preloads it with requests, and logs the simulation output to a file.

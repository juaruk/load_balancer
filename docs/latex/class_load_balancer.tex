\doxysubsection{Detailed Description}
Distributes incoming web requests among a pool of web servers. 

The \doxylink{class_load_balancer}{Load\+Balancer} maintains a queue of incoming requests and a vector of pointers to \doxylink{class_web_server}{Web\+Server} instances. It simulates time passing by clock cycles, distributes queued requests to available servers, and logs the simulation.

Detailed Description

Distributes incoming web requests among a pool of web servers.

The LoadBalancer maintains a queue of incoming requests and a vector of pointers to WebServer instances. It simulates time passing by clock cycles, distributes queued requests to available servers, and logs the simulation.

void LoadBalancer::tick ( )

Advances the simulation by one clock cycle.

Increments current time, ticks each server, generates new requests probabilistically, and distributes them.

Increments the internal clock, calls tick() on all servers, probabilistically adds new requests with a 70% chance, and distributes any queued requests to available servers.
